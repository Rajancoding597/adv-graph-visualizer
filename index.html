🎯 Index Base

          
Choose whether your nodes are 0-based (0, 1, 2, ...) or 1-based (1, 2, 3, ...). Most LeetCode problems use 0-based indexing.

        

        

          
📋 Input Formats

          
    
            
  • Adjacency List: Array of arrays where index = node, values = neighbors
    • 
            
  • Edge List: Array of [u, v] pairs
    • 
            
  • Matrix: Square matrix where matrix[i][j] = 1 means edge i→j
    • 
            
  • Weighted: Array of [u, v, weight] triples
    • 
            
  • Text: Lines like "0-1" or "0->1" or "0-1:5"
